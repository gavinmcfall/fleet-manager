# Theme 03: Erkul Technical

Ultra-clean, high-contrast, data-dense dashboard inspired by erkul.games. Minimal decoration, maximum readability. Every pixel earns its place — no glow effects, no gradients, just crisp data presentation.

## Inspiration
erkul.games ship loadout tool — the gold standard for Star Citizen data tools. Clean, fast, information-dense.

## Special Effects
- **None.** That's the point.
- Zero border-radius (sharp corners throughout)
- Tighter padding than default (stat-cards: p-3, panel-headers: py-2.5)
- Smaller label text (11px) for maximum information density
- Flat glow-line (solid border color, no gradient)
- Faster transitions (150ms vs 200ms)

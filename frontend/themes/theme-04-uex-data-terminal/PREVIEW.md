# Theme 04: UEX Data Terminal

Warm amber/orange data terminal inspired by uexcorp.space. Commodity trading terminal in a space station — the kind of screen you'd stare at while waiting for laranite prices to spike. Dense information, warm tones, professional but lived-in.

## Inspiration
uexcorp.space commodity trading interface — warm amber UI, data-dense layouts, commodity terminal aesthetic.

## Special Effects
- **Warm panel glow:** Subtle amber box-shadow on panels
- **Header gradient:** Left-to-right amber fade on panel headers
- **Text shadow:** Warm amber text-shadow on stat values and accented text
- **Amber glow-line:** Gradient glow-line with amber box-shadow
- **Warm border tint:** Borders use brown-tinted color instead of cool gray
